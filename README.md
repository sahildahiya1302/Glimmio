# Glimmio

This repository contains PHP based web application code. The main working directory is **imx (1)** which holds all backend and frontend sources. Duplicate folders `imx` and `public_html (13)` were removed for clarity.

A fresh `instagram-theme.css` stylesheet provides a modern look inspired by Instagram across all pages. A complementary `dark-theme.css` allows users to toggle dark mode from the header.

See [`imx (1)/README.md`](imx%20(1)/README.md) for a full listing of files and an overview of how components relate.

Recent updates introduce a comment popup on the feed for quick interactions without leaving the page.

For upcoming refactors and security tips, review [IMPROVEMENT_PLAN.md](IMPROVEMENT_PLAN.md).
