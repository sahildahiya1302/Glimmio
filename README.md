# Glimmio

This repository contains the Glimmio dashboard prototype. The `imx` folder holds a PHP based social platform with brand and influencer dashboards. Stylesheets in `imx/css` provide the layout and responsive design used throughout the project.

The main dashboard (`imx/dashboard.php`) now includes a responsive layout that mimics mobile app behaviour with a bottom navigation bar while keeping a sidebar on desktop screens.

Recent updates enhance the feed with a mobile first layout. Posts are displayed in a card grid and can be filtered using a sticky chip bar (all, influencers, brands, polls, videos, carousel or trending). Each card supports carousels, polls and video playback with Instagram‑like reaction buttons and a call to action.

The Ads tab offers a campaign builder for brands and an applications panel for influencers. Wallet APIs track balances, payouts and spending with simple UPI withdrawals. Campaign cards now show status badges (draft, published, live, completed, archived).
